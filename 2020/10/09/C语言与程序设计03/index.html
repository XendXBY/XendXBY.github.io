scanf()

scanf()函数有int型返回值，返回值如下：
若scanf()成功读入数据，则返回值为成功读入的数据个数
若scanf()未读入数据，则返回值为 -1 或 EOF （end of file）
注：在命令行（小黑框）中，按Ctrl+Z后按回车即可结束输入

C语言的基本元素

字符及词法元素

字符集

C源程序由字符序列构成：
英文字母：a ~ z、A ~ Z
数字字符：0 ~ 9
符号字符：*、%、#
空白字符：空格、换行符（回车）、制表符、换页符等

词法元素

称为记号，分为：运算符、关键字、标识符、常量、标点符号（;等）
记号间可以（或必须）用空白字符隔开
编译器总会从左向右依次检索 最长 的记号来组织程序
例：x+++++y
编译器认为：((x++)++) + y

标识符

用户用来标识函数、变量的若干字符（起名字）
命名规则：
1.不能为C语言已有的关键字
2.由字母、数字、下划线组成，数字不能放在开头

关键字

被系统赋予特定含义且有特定作用的单词

基本数据类型

分为基本类型、导出类型、空类型（void）

字符型

分类：char
长度：1B（-128~127）
储存方式：ASCII码
实质：整型
在储存数值不大的情况下，可以用字符型代替整型
例：

输出：97, a

整型

分类：short (int)，int，long (int)，long long (int)
长度：与编译器有关（short为2B）
注：括号表示可以省略

浮点型

分类：float，double，long double，long long double
表示：
先将小数转换为二进制：整数部分同整数，小数部分：乘2取整法
浮点数分为三部分：符号位、尾数、指数
下文来自CDSN：
根据国际标准IEEE（电气和电子工程协会）规定，任何一个浮点数NUM的二进制数可以写为：
$NUM = (-1) ^ S * M * 2 ^ E$;//(S表示符号，E表示阶乘，M表示有效数字)
①当S为0时，表示一个正数；当S为1时，表示一个负数
②M表示有效数字，1<= M <2
③2^E表示指数
比如十进制的3.0，二进制就是0011.0 就可以写成（-1）^ 0 * 1.1 * 2 ^ 1
在比如十进制的-3.0，二进制就是-0011.0 就可以写成（-1）^ 1 * 1.1 * 2 ^ 1
而规定float类型有一个符号位（S），有8个指数位（E），和23个有效数字位（M）
double类型有一个符号位（S），有11个指数位（E），和52个有效数字位（M）
IEEE对于（有效数字）M和（指数）E有特殊的规定： （以float为例）
1.因为M的值一定是1<= M <2，所以它绝对可以写成1.xxxxxxx的形式，所以规定M在存储时舍去第一个1，只存储小数点之后的数字。这样做节省了空间，以float类型为例，就可以保存23位小数信息，加上舍去的1就可以用23位来表示24个有效的信息。
2.对于E（指数）E是一个无符号整数所以E的取值范围为（0~ 255），但是在计数中指数是可以为负的，所以规定在存入E时，在它原本的值上加上中间数（127），在使用时减去中间数（127），这样E的真正取值范围就成了（-127~128）。
对于E还分为三种情况：
①E不全为0，不全为1:
这时就用正常的计算规则，E的真实值就是E的字面值减去127（中间值)，M的值要加上最前面的省去的1。
②E全为0
这时指数E等于1-127为真实值，M不在加上舍去的1，而是还原为0.xxxxxxxx小数。这样为了表示0，和一些很小的整数。
所以在进行浮点数与0的比较时，要注意。
③E全为1
当M全为0时，表示±无穷大（取决于符号位）；当M不全为1时，表示这数不是一个数（NaN）
看不懂，了解一下，开心就好
根据储存原理可知，浮点数储存有误差（尤其是无限循环小数等）

无符号

默认数据类型为有符号（等价于在数据类型前加signed关键字）
在数据类型前加unsigned关键字，表示该变量为无符号数，即该变量只能储存正数
这样做会让变量储存数据的范围/精度扩大一倍

常量与变量

常量

直接写出来的数就是常量

整型常量

表示方法：
二进制：C语言不支持
十进制：无前缀
八进制：前缀为0
十六进制：前缀为0x或0X（0是数字不是字母O）
例：23可表示为027或0x17
常量后缀：
u/U：unsigned
l/L：long
ul/UL：unsigned long
ll/LL：long long
ull/ULL：unsigned long long

浮点型常量

表示方法：
十进制形式：23.2
指数形式：2.5e3 = 2500
.23e2 = 23

字符常量

表示方法：
单引号 引起来的一个字符（转义字符可能有多个）
‘\n’：回车
‘\t’：制表符
‘\0’：空字符
‘\?’：问号
‘\‘’：单引号
‘\“‘：双引号
‘\\‘：一个反斜线

数字转义序列

整体表示ASCII码对应的字符
‘\ooo’：最多3个8进制数字
‘\xhh’：最多2个16进制数字
eg：’\101’ == ‘\x41’ == ‘A’
‘\t’ == ‘\11’ == ‘\011’ == ‘\x9’ == ‘\x09’ == ‘\t’

字符常量整体与字符对应的ASCII码值等价

eg：’a’ == 97

字符串常量

表示方法：双引号引起来的0或多个字符
“”：空字符串
“Hi\n”：包含三个字符

将一个字符串写到多行

行连接：
“Hello,\
world!”
字符串连接：
{
“Hello”
“world!”
}

字符串结束标识

字符串会在末位加’\0’（空字符，ASCII码值为0）
“Hello” == ‘H’, ‘e’, ‘l’, ‘l’, ‘o’, ‘\0’
‘H’, ‘e’, ‘\0’, ‘l’, ‘l’, ‘o’ == “He”
eg：求字符串的长度

输出：12
