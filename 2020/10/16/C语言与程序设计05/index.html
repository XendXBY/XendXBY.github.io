逗号表达式

e1, e2
先计算e1，再计算e2，表达式的值与e2相同
eg：x = i = 4, i % 3
表达式的值为1
eg：写一个函数，代替scanf("%d", &x);的功能

位运算

&：按位与 |：按位或
^：按位异或（半加，即不进位加法） ~：按位取反
eg：01100110 & 11110000 = 01100000
01100110 | 11110000 = 11110110
01100110 ^ 11110000 = 10010110
~01100110 = 10011001
注：按位取反操作对符号位同样适用，但具体数值与数据类型有关
eg：char a = 102(01100110)
~a == -25(10011001)
unsigned char b = 102(01100110)
~b == 153(10011001)
eg：判断一个数是奇数还是偶数

eg：大小写转换

<<：左移 >>：右移
（<<<：无符号左移 >>>：无符号右移）
e << n == $e * 2^n$
e >> n == $e / 2^n$
注：右移时若数据为有符号类型，则最高位可能补0或补符号位，因此一般需要右移的数据一般声明为无符号类型
eg：x = 00110101
x << 1 == 01101010
x >> 1 == 00011010
注：移位操作相等于加减运算，不会改变原变量的值
eg：x = 00110101
x << 1
（此时x == 00110101）
x = x << 1
（此时x == 01101010）
eg：设二进制整数右起分别为第1位，第2位，第3位……
则：将整数x的第m位起的右n位取出来
