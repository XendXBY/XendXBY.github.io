客户端必备组件

开始之前，您应确认已在要运行Hyperledger Fabric的平台上安装了以下所有必备组件。

安装Git

如果尚未安装最新版本的git，或者如果您在运行curl命令时遇到问题，请下载它。

安装cURL

如果尚未安装cURL工具的最新版本，或者从文档中运行curl命令时遇到错误，请下载它。

Hyperledger Fabric智能合约（链码）API

Hyperledger Fabric提供了许多API，以支持以各种编程语言开发智能合约（链码）。智能合约API可用于Go，Node.js和Java：

Go contract-api。
Node.js合约API和Node.js合约API文档。
Java合约API和Java合约API文档。

Hyperledger Fabric应用程序SDK

Hyperledger Fabric提供了许多SDK，以支持使用各种编程语言开发应用程序。SDK可用于Node.js和Java：

Node.js SDK和Node.js SDK文档。
Java SDK和Java SDK文档。

使用SDK开发的前提条件可以在Node.js SDK README和Java SDK README中找到。

此外，还有两个尚未正式发布的应用程序SDK（针对Python和Go），但仍可供下载和测试：

Python SDK。
Go SDK。

当前，Node.js，Java和Go支持Hyperledger Fabric v1.4中提供的新应用程序编程模型。

Hyperledger Fabric CA

Hyperledger Fabric提供了可选的 证书颁发机构服务 ，您可以选择使用该服务来生成证书和密钥材料，以配置和管理区块链网络中的身份。但是，可以使用任何可以生成ECDSA证书的CA。
