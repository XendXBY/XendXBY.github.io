77.组合

发表于 2020-09-08 更新于 2020-09-27 分类于 力扣

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/combinations

题目描述

给定两个整数 n 和 k，返回 1 … n 中所有可能的 k 个数的组合。

样例

输入: n = 4, k = 2
输出:
[
[2,4],
[3,4],
[2,3],
[1,2],
[1,3],
[1,4],
]
